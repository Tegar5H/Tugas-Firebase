# **App Name**: TaskinAja

## Core Features:

- User Authentication: Secure user login and registration using Firebase Authentication (email & password).
- Task Management: Create, read, update, and delete (CRUD) tasks with title, description, deadline, status, and labels.
- Task Filtering & Sorting: Filter tasks by status (completed/incomplete) and sort by deadline or creation date.
- Dashboard: Display an overview of today's tasks, upcoming deadlines, and task completion status.
- Data Persistence: Store and retrieve user task data in Firestore, ensuring data isolation per user.
- Intelligent task suggester: Leverage AI to provide users with suggestions for task labels based on title and descriptions entered using a tool. These generated labels aid the user in quickly categorizing, searching and filtering.

## Style Guidelines:

- Primary color: Light blue (#64B5F6) for a calm and focused feel.
- Background color: Very light blue (#E3F2FD).
- Accent color: Orange (#FFB74D) to highlight important actions and deadlines.
- Body and headline font: 'Inter' (sans-serif) for a modern, readable interface.
- Use minimalist and clear icons for task actions and categories.
- A clean, responsive layout with a focus on task prioritization and ease of use on both desktop and mobile devices.
- Subtle transitions and animations to provide feedback on user actions, such as task completion.
